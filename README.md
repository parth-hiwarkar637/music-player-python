# Python Music Player 🎵

A simple desktop music player built with Python.
This project was created as part of my journey learning Python by building small, practical applications.

The program allows users to load and play music files from their computer through a lightweight graphical interface.

---

## Features

* Play and pause music
* Load songs from your local system
* Simple graphical interface
* Lightweight and easy to run

---

## Technologies Used

* **Python**
* **Tkinter** – for the graphical user interface
* **Pygame** – for audio playback

---



## Installation

1. Clone this repository:

```
git clone https://github.com/parth-hiwarkar637/music-player-python.git
```

2. Move into the project folder:

```
cd music-player-python
```

3. Install the required dependencies:

```
pygame
tkinter
```

---

## Running the Program

Run the following command:

```
python music_player.py
```

After running the program, the music player window will open and you can load and play music files from your system.
